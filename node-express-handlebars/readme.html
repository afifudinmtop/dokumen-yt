Apa yang Kita Pelajari dari Contoh Handlebars

  1. Pengaturan Handlebars dengan Express:
    • Integrasi Handlebars dengan aplikasi Express menggunakan express-handlebars.
    • Pengaturan Handlebars sebagai mesin view dengan app.set('view engine', 'handlebars').
  2. Penggunaan Layouts dan Views:
    • Membuat layout dasar dengan Handlebars (main.handlebars) yang digunakan sebagai kerangka untuk halaman web.
    • Penyisipan konten dari view spesifik dengan {{{body}}} dalam layout utama.
  3. Penggunaan Partials:
    • Membuat komponen yang dapat digunakan kembali dengan partials (seperti navbar.handlebars).
    • Memungkinkan penggunaan ulang bagian kode di berbagai tempat dalam aplikasi.
  4. Penanganan Konteks dan Variabel:
    • Mengirim data dari server ke template Handlebars melalui objek konteks.
    • Menampilkan variabel dengan sintaks {{ }}.
  5. Kondisional dan Looping:
    • Penggunaan blok {{#if}}...{{/if}} untuk konten kondisional.
    • Penggunaan blok {{#each}}...{{/each}} untuk mengulang melalui daftar item.
  6. HTML Escaping:
    • Penanganan otomatis karakter escape oleh Handlebars untuk mencegah serangan XSS.
    • Penyajian konten HTML mentah yang tidak di-escape menggunakan {{{ }}}.
  7. Ekspresi dan Properti:
    • Menggunakan ekspresi untuk mengakses properti dari konteks yang dikirim dari server.
  8. Pengujian Aplikasi:
    • Menjalankan server dengan Node.js dan melihat hasil di browser.
